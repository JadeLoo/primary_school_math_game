# 图形算式大冒险 — 用户手册

## 简介

这是一款面向小学一年级学生的数学闯关游戏，通过"图形算式"训练计算能力。孩子需要根据算式中图形符号与数字的关系，推算出每个图形代表什么数字。

游戏包含 **6 种题型、18 个关卡、6 大主题区域**，通关即可收集奖品。

---

## 如何打开

1. 在浏览器地址栏输入游戏网址（例如 `http://localhost:5173`）
2. 推荐在**平板**或**手机**上使用，方便孩子触摸操作
3. 支持 Chrome、Edge、Safari 等主流浏览器

## 游戏流程

### 1. 首页

打开游戏后看到标题"图形算式大冒险"，点击 **「开始冒险」** 进入关卡地图。

### 2. 冒险地图

地图上排列着 6 个主题区域，每区 3 关，共 18 关：

| 区域 | 关卡 | 主要题型 | 难度 |
|------|------|----------|------|
| 🌱 数字草原 | 1-3 | 单一图形求值、图形替代数字 | 数字 ≤ 15 |
| 🌲 图形森林 | 4-6 | 多图形联立 | 数字 ≤ 20 |
| 🏔️ 推理山丘 | 7-9 | 链式推理 | 数字 ≤ 24 |
| 🌊 加法河流 | 10-12 | 竖式进位加法 | 两位数 |
| 🏰 减法城堡 | 13-15 | 竖式退位减法 | 两位数 |
| ⭐ 综合星空 | 16-18 | 全部题型混合 | ≤ 100 |

- **🔒 锁住**的关卡需要通关前一关才能解锁
- **★ 星星**表示该关历史最佳成绩（最高三星）
- 带 **👑 皇冠**的是 Boss 关，难度稍高
- 顶部可看到总星数，点击 **「🎁 奖品册」** 查看收集进度

### 3. 答题界面

点击已解锁的关卡进入答题。

**操作方式：**

- **先选框**：点击你要填的答案框（蓝色光晕表示已选中）
- **再按键**：点数字键盘输入数字，最多输入两位数
- **⌫ 删除**：删除当前选中框的最后一位
- **确认**：全部填好后点击「确认」
- **🔊 再读一遍**：点击喇叭图标重播题目语音

**进度条：** 顶部 3 个圆点显示当前进度（● = 已完成，◌ = 未做）

**反馈：**
- 答对 → 绿色 ✓，连击计数 +1
- 答错 → 红色 ✗，显示正确答案，连击中断
- 连续答对 3 题触发庆祝动画

### 4. 通关结果

完成 3 道题后进入结果页：

| 答对 | 星星 | 结果 |
|------|------|------|
| 3 / 3 | ★★★ | 通关，解锁下一关，收集奖品 |
| 2 / 3 | ★★ | 通关，解锁下一关，收集奖品 |
| 1 / 3 | ★ | 通关，解锁下一关，收集奖品 |
| 0 / 3 | — | 不通过，需重试 |

按钮说明：
- **再玩一次**：重新挑战本关（可刷更高星级）
- **下一关 →**：进入下一关
- **返回地图**：回到冒险地图

## 六种题型说明

### 题型 1：单一图形求值

同一个图形符号出现多次，根据等式求它的值。

> 示例：`★ + ★ + ★ = 9`
> 答案：★ = 3

### 题型 2：多图形联立

两个不同图形，给出两个等式（一个加法、一个减法），联立求解。

> 示例：`▲ + ■ = 12`，`▲ - ■ = 4`
> 答案：▲ = 8，■ = 4

### 题型 3：链式推理

两个图形的值之间有关系，需要一步步推导。

> 示例：`● + ● = ▲`，`▲ + ● = 9`
> 答案：● = 3，▲ = 6

### 题型 4：图形替代数字

直接告知每个图形代表哪个数字，计算表达式的值。

> 示例：`● = 3，▲ = 5，● + ▲ = ?`
> 答案：8

### 题型 5：竖式进位加法

竖式中某些数位被替换为图形符号，需根据竖式计算规则求出图形代表的数字。

> 示例：
> ```
>   ▲ 7
> + 2 ■
> ——————
>   6 4
> ```
> ▲ = 3，■ = 7（37 + 27 = 64）

### 题型 6：竖式退位减法

与题型 5 类似，但为减法且需要退位。

> 示例：
> ```
>   5 ▲
> - 1 8
> ——————
>   3 4
> ```
> ▲ = 2（52 - 18 = 34）

## 常见问题

### Q：进度会丢失吗？
A：游戏进度保存在浏览器本地存储中。只要使用**同一个浏览器**、不清理浏览数据，进度就会保留。

### Q：可以换设备玩吗？
A：目前进度不跨设备同步。如需切换设备，需在新设备上重新开始。

### Q：怎么重置所有进度？
A：在浏览器中清除该网站的本地存储（localStorage），或联系家长操作。

### Q：数字键盘太小了？
A：游戏已针对手机和平板优化布局，数字键盘按钮大小适合儿童手指点击。如仍嫌小，可旋转设备至横屏模式。

### Q：没声音怎么办？
A：朗读功能使用浏览器内置的语音合成引擎。请确保：
1. 设备音量已打开
2. 浏览器未被静音
3. 系统语言包含中文语音包（大部分设备已自带）

## 家长指南

### 学习建议

1. **从简单的开始**：建议按关卡顺序推进，不要跳关
2. **适可而止**：每次玩 1-2 关（3-6 题），避免长时间盯屏幕
3. **鼓励为主**：即使得 1 星也算通关，重点培养兴趣
4. **亲子互动**：低年级孩子识字有限，前几关建议家长在旁边辅助读题
5. **草稿纸辅助**：遇到竖式题，可以给孩子一张草稿纸在旁演算
